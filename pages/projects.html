API de Envio de E-mails

BackEnd | API

Esta é uma API simples para envio de e-mails construída com Node.js e Express, utilizando o Nodemailer para gerenciar o envio de mensagens. Feita para envio dos dados do Form do Portfolio.
Recebe dados de formulário (nome, email, mensagem) Envia e-mails usando SMTP (Gmail ou outro provedor) Gerenciamento de erro e sucesso Suporte a CORS para integração com frontend
